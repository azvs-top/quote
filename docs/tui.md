# TUI 文档

`quote-tui` 提供基于终端的浏览界面，适合快速分页查看与详情检查。

## 启动方式

```bash
# 开发构建后运行
cargo run --bin quote-tui

# 或 release 二进制
./target/release/quote-tui
```

说明：
- 启动时会加载与 CLI 相同的应用配置（数据库、存储）。
- 首次进入会自动加载总数与第一页数据。
- 若加载失败，状态栏会显示错误信息，可按 `r` 重试。

## 界面结构

- 顶部标题栏：显示 `total/page/limit/items`。
- 正文区：
  - 列表页：左侧是 quote 列表，右侧是选中项预览。
  - 详情页：全屏展示选中 quote 的完整字段内容，可滚动。
  - 帮助页：显示键位与命令速查。
- 底部状态栏：显示当前状态或命令输入（以 `:` 开头）。

## 键位（普通模式）

### 列表导航

- `j` / `↓`：下移一项
- `k` / `↑`：上移一项
- `J`：跳到当前页最后一项
- `K`：跳到当前页第一项
- `h` / `←`：上一页
- `l` / `→`：下一页
- `H`：跳到第一页
- `L`：跳到最后一页
- `Enter`：打开选中项详情页
- `r`：重新加载总数和当前页数据
- `q`：退出

### 详情页

- `j` / `↓`：向下滚动
- `k` / `↑`：向上滚动
- `:`：进入命令模式
- `q`：退出

## 命令模式

按 `:` 进入命令模式，按 `Enter` 执行：

- `:help`：打开帮助页
- `:list`：返回列表页
- `:q` 或 `:quit`：退出程序

编辑行为：
- `Backspace`：删除一个字符
- `Esc`：取消命令输入并回到普通模式

## 数据展示规则

- 列表项显示：`id + inline 预览文本`。
- inline 预览优先级：`en` -> `zh` -> 第一个可用语言。
- 详情页会显示：
  - `id` / `remark`
  - `inline` 多语言文本
  - `external` 多语言对象 key
  - `markdown` 多语言对象 key
  - `image` 对象 key 列表（含索引）

## 与 CLI / HTTP 的边界

- `quote-tui` 当前聚焦“浏览与查看”，不提供创建/更新/删除操作。
- 数据写入与对象下载请使用 CLI（`quote create/update/delete/download`）。
- HTTP 入口当前仍为占位，暂无可用 API。
